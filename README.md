# Jogo Termo - Java 🧩

Clone do jogo Termo desenvolvido em Java, apresentando funcionalidades exclusivas e focado na aplicação prática de princípios de Programação Orientada a Objetos (POO).

🚀 Funcionalidades
- Lógica de verificação de letras (posições corretas e incorretas).
- Manipulação de Strings e Matrizes.
- Interface Gráfica.

🛠️ Tecnologias
- Java JDK 17+
- Lógica de Programação Avançada


⚙️ Como Executar
- make clean
- make
- make run
  
Desenvolvido por Álefe Tonidandel durante a disciplina de Programação Oriantada a Objetos da UFOP.
